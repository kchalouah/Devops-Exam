# DevOps Exam Project - Simple Todo App

This repository contains the source code and DevOps configuration for the "ChuZone" digital product proof of concept.

## Structure

- `app/`: Frontend React Application (Simple Todo App).
- `.github/workflows/`: GitHub Actions for CI/CD.
- `terraform/`: Infrastructure as Code (AWS).
- `k8s/`: Kubernetes Manifests for GitOps.
- `scripts/`: Helper scripts for K8s setup.

## Prerequisites

- Docker
- Terraform
- GitHub Account with Secrets configured (`DOCKERHUB_USERNAME`, `DOCKERHUB_TOKEN`).
